## Local Setup (Create your own .env file and add your variables in it)

```sh
$ git clone https://github.com/itzadhi/urlshortener-backend.git
```

```sh
$ cd urlshortener-backend
```

```sh
$ npm install
```

```sh
$ npm run server
```
